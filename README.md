# TrendyWearRetailers
TrendyWearRetailers' very own e-commerce website. Created using Next.js and Supabase backend.

Make sure you have Node.js and NPM up to date

Instructions:
1. Clone the project (Recommend Github Desktop)
2. Open project in VS code.
3. Run Command => cd trendywear
4. Run Command => npm install
5. Run Command => npm run dev
